# Archivos de Sonido del Juego

Esta carpeta debe contener los siguientes archivos de audio para los efectos de sonido del juego:

## Archivos Requeridos

| Archivo | Descripción | Duración Recomendada |
|---------|-------------|---------------------|
| `move.mp3` | Sonido sutil de movimiento (tipo "whoosh" corto) | ~100ms |
| `line-clear.mp3` | Sonido de destrucción de línea (tipo "pop" satisfactorio) | ~300ms |
| `wall-hit.mp3` | Sonido de colisión contra muro (tipo "thud" seco) | ~150ms |
| `piece-lock.mp3` | Sonido de pieza colocada correctamente (tipo "click" suave) | ~200ms |

## Dónde Obtener Sonidos Gratuitos

1. **Pixabay** - https://pixabay.com/sound-effects/
   - Buscar: "whoosh", "pop", "click", "thud"
   - Licencia: Libre de regalías

2. **Freesound** - https://freesound.org/
   - Requiere cuenta gratuita
   - Verificar licencia de cada sonido

3. **Mixkit** - https://mixkit.co/free-sound-effects/
   - Licencia: Libre para uso comercial

## Formato

- Formato: MP3 (recomendado) o WAV
- Calidad: 128kbps o superior
- Mono o Stereo

## Nota

El juego funcionará sin estos archivos, pero no reproducirá efectos de sonido hasta que se agreguen.
